L4InvalidBraceClauseException

Defines the name for the invalid brace clause exception 
NSString *const L4InvalidBraceClauseException;

Some of the format specifiers can be followed by content surrounded by braces ({}). When this brace clause is invalid for any reason, the L4InvalidBraceClauseException is thrown
