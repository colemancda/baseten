Defined Types


L4RollingFrequency

The accepted constants for L4DailyRollingFileAppenderFrequency's setFrequency: method. 
typedef enum L4RollingFrequency
    {
    never,
    monthly,
    weekly,
    daily,
    half_daily,
    hourly,
    minutely
} L4RollingFrequency;

Constants

never
Never roll the file over (this is a default setting)
monthly
Roll the file over every month
weekly
Roll the file over every week
daily
Roll the file over every day
half_daily
Roll the file over every half day (once at midnight, once at noon)
hourly
Roll the file over every hour
minutely
Roll the file over every minute
