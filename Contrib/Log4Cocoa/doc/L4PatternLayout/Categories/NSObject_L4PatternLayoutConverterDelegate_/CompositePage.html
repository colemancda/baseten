Parameter Descriptions
token
The token string to convert
logEvent
An L4LoggingEvent that contains information about the current logging request
convertedString
A reference to an NSString that will contain the result of the token string's conversion upon exiting the method
method result
Return YES to indicate that you have converted the token string. Return NO to let the pattern layout's default behavior attempt to convert it.
